# Rilog App

**Rilog App** на [rilog.online](https://www.rilog.online) — це дашборд, куди потрапляють всі ваші зафіксовані події. Це веб-застосунок — не потребує встановлення, лише браузер.

## Що ви можете робити в застосунку

- **Створювати та керувати застосунками** — логічні контейнери для ваших проєктів, кожен зі своїм connection key.
- **Переглядати з'єднання** — кожен запущений екземпляр вашого застосунку з встановленою rilog-lib з'являється як з'єднання.
- **Переглядати події в реальному часі** — бачте HTTP-запити, помилки, перегляди сторінок і кастомні події в міру їх виникнення.
- **Фільтрувати події** — за кодом статусу, HTTP методом, типом події, часовим діапазоном тощо.
- **Переглядати деталі подій** — розгортайте будь-яку подію, щоб побачити повні заголовки та тіла запитів/відповідей у форматованому code view.
- **Переглядати аналітику** — показники помилок, успішності запитів, найбільш проблемні endpoint, трафік з часом.
- **Ділитися подіями** — генеруйте публічне посилання на будь-який snapshot події для звітів про баги або обговорення в команді.
- **Залишати коментарі** — анотуйте події нотатками для вашої команди.

## Навігація

Застосунок структурований за трьома рівнями:

```
Акаунт
  └── Applications (ваші проєкти)
        └── Connections (запущені екземпляри)
              └── Events (окремі записи логів)
```

Використовуйте ліву бічну панель для навігації між розділами. Верхній breadcrumb показує вашу поточну позицію.

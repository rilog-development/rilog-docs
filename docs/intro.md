# Що таке Rilog?

**Rilog** — це легка платформа спостережуваності для веб-застосунків. Вона складається з двох частин, що працюють разом:

- **rilog-lib** — JavaScript/TypeScript [бібліотека](https://www.npmjs.com/package/@rilog-development/rilog-lib), яку ви встановлюєте у свій застосунок
- **Rilog App** — веб дашборд на [rilog.online](https://www.rilog.online), де ви переглядаєте всі зібрані дані

Після підключення Rilog автоматично фіксує HTTP-запити й відповіді (fetch автоматично, axios — через додаткові interceptors), кліки на кнопках і посиланнях, виклики `console.error` і `console.warn`. Ви бачите все в реальному часі в дашборді — без написання власної backend-інфраструктури.

## Чому Rilog?

Традиційні інструменти моніторингу або надто складні (потребують агентів, налаштування інфраструктури або DevOps-експертизи), або надто прості (фіксують лише необроблені виключення). Rilog знаходиться посередині: простий у налаштуванні, але достатньо потужний, щоб дати вам реальні відповіді, коли щось іде не так у production.

**Головна проблема, яку він вирішує:** За допомогою Rilog App ви можете дізнатися про помилки в веб застосунках, раніше ніж вам про це повідомлять користувачі чи тестувальники. Додатково ви можете прослідкувати весь шлях, який пройшов користувач до виникнення того чи іншого багу.

## Основні поняття

### Application (Застосунок)

Ви створюєте застосунок у дашборді, отримуєте API ключ і використовуєте його в `rilog-lib`.

### Connection (З'єднання)

**Connection** — це один запущений екземпляр вашого застосунку, підключеного до Rilog. Зʼєднання діє в межах одного відкритого браузера. Кожен користувач вашого застосунку буде мати своє унікальне зʼєднання.

### Event (Подія)

**Event** — це будь-яка зафіксована подія: HTTP-запит, клік по кнопці, console-повідомлення або кастомне debug-повідомлення.

## Огляд архітектури

```
Ваш веб-застосунок
  └── rilog-lib (встановлено як npm пакет)
        ├── Відстежує різні типи подій
        └── Надсилає події → Rilog backend API

Rilog backend
  └── Зберігає події, будує аналітику

Rilog App (rilog.online)
  └── Відображає події, фільтри, аналітику, sharing
```
